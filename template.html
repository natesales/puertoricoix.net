Policy

If a member causes issues for the exchange or other members, the offending port may be brought down until the issue is corrected.

  • Only announce IP space that you're authorized to announce.
  • Only 0x0800 (IPv4), 0x0806 (ARP), and 0x86dd (IPv6) ethertypes are permitted.
  • Only one MAC address per port without prior authorization.

Services

Google Global Cache

PR-IX has a Google Global Cache node to cache common Google content. It announces 23.171.112.32/27 and 2607:9280:1:e100::/64 to the route servers, originated by AS36040.

Notable DNS Servers

  • D and E roots (hosted by PCH AS42)
  • Quad9 Recursive Resolver (hosted by PCH AS42)
  • AS112 Blackhole Server

Route Servers

The PR-IX route servers use AS18804 on the following addresses:

  • 204.138.0.251
  • 2620:102:b000::251
  • 204.138.0.252
  • 2620:102:b000::252

The route servers run enforce prefix filters for each peer.
